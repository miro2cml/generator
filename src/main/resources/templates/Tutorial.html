How to get started

The first thing you need to do is authenticate yourself with Miro. Afterwards your boards will appear in the gray list. Choose your Board Type and select your MiroBoard you like to convert, as shown in the figure.



With the Button Map Board, you can convert your board. When the conversion is finished a new field appears: Context Mapper Ouput. If the first section is green, the input board was perfect. In the figure you can see such an ouput. If the first section is orange, there are some small errors. If it is red, the board could not be converted. In the log file as well as in the colored section are the information what went wrong. The preview shows the CML output or the log File if something went wrong. The files can be downloaded via the buttonn Download Logfile and Download CML.



Board Type: EducatedGuessed

There is a possibility to set the board type to EducatedGuessed. The converter then decides which template was used and maps the board accordingly.
